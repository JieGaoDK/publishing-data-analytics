# Publishing Data Analytics Project

This project simulates editorial workflow analytics for a scientific journal.

The goal is to practice data analysis skills relevant to academic publishing.

## Project Goals

- Analyze editorial workflow metrics
- Calculate publishing KPIs
- Build data visualization dashboards
- Practice SQL and Excel analysis

## Skills Practiced

- Excel Pivot Tables
- SQL queries
- Data visualization
- Publishing metrics analysis

## Simulated Journal Workflow

Submission  
Editor assignment  
Reviewer invitation  
Peer review  
Editorial decision  
Production  
Publication

## Tools Used

Excel  
SQL  
Tableau Public
